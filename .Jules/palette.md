## 2024-05-18 - German Localization Pattern
**Learning:** The application uses German localization for its UI elements and accessibility features, meaning all added ARIA labels and title tooltips must be in German (e.g., 'Abspielen', 'Herunterladen').
**Action:** Always inspect the surrounding component code or UI text to determine the correct language and apply localized text for a11y improvements, maintaining consistency.
