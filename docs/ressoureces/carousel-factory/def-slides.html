A: Classic – Term + Definition mit Border

Begriff erklärt
Prompt Engineering

Die Kunst, Anweisungen so zu formulieren, dass ein KI-Modell das tut, was du wirklich brauchst – nicht das, was du wörtlich gesagt hast.

B: Minimal – Zentriert, ohne Border

Kurz erklärt
Token

Die kleinste Einheit, die ein Sprachmodell verarbeitet. Etwa 3–4 Buchstaben oder ein kurzes Wort.

Kurz erklärt

Die Menge an Text, die ein Modell gleichzeitig „sehen" kann. Je größer, desto mehr Kontext möglich.

C: Mit Beispiel – Definition + Praxis

Begriff erklärt
System Prompt

Eine versteckte Anweisung, die das Verhalten des Modells für die gesamte Konversation festlegt.

Beispiel

„Du bist ein freundlicher Assistent, der immer in einfacher Sprache antwortet."

RL.
03
Begriff erklärt
Few-Shot Prompting

Eine Technik, bei der du dem Modell Beispiele gibst, bevor du die eigentliche Aufgabe stellst.

Beispiel

„Übersetze: Hund → Dog, Katze → Cat, Haus → ?"

D: Abgrenzung – Was es ist vs. was nicht

Begriff erklärt
Prompt Engineering
Was es ist

Die strukturierte Formulierung von Anweisungen, um bessere Ergebnisse zu erzielen.

Was es nicht ist

Geheime Tricks oder Zauberformeln. Auch kein Ersatz für klares Denken.

RL.
03
Begriff erklärt
KI-Agent
Was es ist

Ein System, das eigenständig Aufgaben plant, ausführt und Ergebnisse überprüft.

Was es nicht ist

Ein Chatbot mit mehr Features. Oder ein autonomes System ohne menschliche Kontrolle.

E: Glossar – Mehrere Begriffe kompakt

Glossar
LLM

Large Language Model – ein KI-Modell, das auf riesigen Textmengen trainiert wurde.

Token

Die kleinste Texteinheit, die das Modell verarbeitet. Etwa 3–4 Zeichen.

Inference

Der Prozess, bei dem das Modell eine Antwort generiert.

RL.
03
Prompting-Begriffe
Zero-Shot

Prompt ohne Beispiele – das Modell soll direkt antworten.

Few-Shot

Prompt mit 2–5 Beispielen, die das gewünschte Format zeigen.

Chain-of-Thought

Technik, bei der das Modell seinen Denkprozess Schritt für Schritt zeigt.
